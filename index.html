Configurações

Tags utilizadas para configurar o documento:
html:5 -> snipet para configurar o documento de acordo com o html 5
head -> Tudo dentro do head carrega antes da página iniciar
body -> Aqui fica os elementos que serão exibidos visualmente para o usuário
title -> Título da página, aparece na aba

Elementos

button -> Adiciona button
input -> Adiciona formulários

h1, h2...h6

Tag de Títulos e subtítulos que vai do maior título(h1) até o menor (h6)

Tags de texto

b->Tag para deixar o texto em negrito
strong -> Tag para deixar o texto negrito
mark -> Adicionar um marca texto no texto
p -> delimitar paragrafos
u -> adicionar sublinhado ao texto
a -> adicionar hyperlink
blockquote -> adicionar citação

Tags de lista

ol-> Utilizado para listas ordenadas
ul-> Utilizado para listas não ordenada
li-> Delimita quem são os itens da lista

Atributos

type -> Defini o tipo da tag, pode ser usada em varias tags - ol, input;
src -> Caminho de um arquivo, bastante usado em img;
href -> caminho de referência, usado principalmente em ancoragem;
